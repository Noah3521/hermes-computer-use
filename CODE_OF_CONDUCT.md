# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

TL;DR — be excellent. Harassment, discrimination, and personal attacks are not welcome. Discussions should focus on the technical merits and user impact of the work, not on contributors.

Enforcement contact: see the maintainer email in `pyproject.toml`.
